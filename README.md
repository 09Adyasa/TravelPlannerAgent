\# 🌍 AI Travel Planner Agent



An AI-powered Travel Planner Agent built using IBM watsonx.ai and Streamlit.



\## ✨ Features



\- 🤖 AI-generated personalized travel itineraries

\- 🌍 Destination recommendations

\- 🏨 Accommodation suggestions

\- 🚗 Transportation recommendations

\- 💰 Budget estimation

\- 🎒 Packing checklist

\- ☀ Weather-aware travel planning

\- 📍 Google Maps integration

\- 📄 PDF itinerary download

\- 🎨 Interactive Streamlit interface



\## 🛠 Technologies Used



\- Python

\- Streamlit

\- IBM watsonx.ai

\- IBM Granite / Llama Model

\- FPDF2

\- Pandas



\## 🚀 How to Run



1\. Clone the repository



```bash

git clone <repository-url>

```



2\. Install dependencies



```bash

pip install -r requirements.txt

```



3\. Create a `.env` file with:



```text

IBM\_API\_KEY=YOUR\_API\_KEY

IBM\_PROJECT\_ID=YOUR\_PROJECT\_ID

IBM\_URL=YOUR\_IBM\_URL

MODEL\_ID=YOUR\_MODEL\_ID

```



4\. Run the application



```bash

streamlit run app.py

```



\## 📌 Project Objective



This project was developed as an AI-powered Travel Planner Agent that generates personalized travel plans based on user preferences such as destination, budget, travel style, weather, and duration.



It demonstrates the use of Large Language Models (LLMs) to automate intelligent travel planning.
